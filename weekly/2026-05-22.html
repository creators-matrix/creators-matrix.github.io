全站改版：从散装页面到双核矩阵

2026-05-22

这个周末干了件大事——把三个散装的独立页面重构为统一的"AI技术工坊 + 量化中医实验室"双核结构。

做了什么

  • 统一全站配色为黑底极客风（Stone 950色系）——之前AI页面白底、中医页面宣纸色，放一起完全不搭
  • 总大门改为双核导航卡片：TECH WORKSHOP / QUANT TCM 并列展示
  • 添加 GDPR Cookie 同意弹窗（localStorage缓存用户选择）
  • 所有工具页升级为"上半屏交互 + 下半屏SEO长文"缝合结构
  • 五运六气推算器从 blog 迁移到独立的 yunqi 工具目录
  • 所有路径改为相对路径（../../模式），确保任意子目录独立部署都能秒开

踩了什么坑

  1. 最坑的：工具页从 blog/ 迁移到 articles/ 后，20篇文章的导航链接全部404。因为旧的 relative path 是 ../../blog/article1.html，移动后没改。查了一个小时才发现。
  2. 算法bug：五运六气的天干化运映射写错了——用了天干自身五行（甲乙木、丙丁火...）而不是天干化运（甲己化土、乙庚化金...）。10个映射错了8个。感谢一位中医背景的读者私信指出。
  3. Pagefind困境：想加静态全文搜索，但 Pagefind 需要先 build 生成索引文件。这和"零构建工具"的原则冲突，暂时在每个文章列表页留了 HTML 占位框。

接下来的计划

  • 给每个工具页补上 "为什么做这个工具" 的作者注记
  • 建一个老码农周报页面（就是这里）
  • 建一个支持/打赏页面
  • 学一下 Tailwind 的响应式断点，把移动端体验优化一下
