AI工具三件套上线

2026-05-15

这周把三个AI相关的小工具写完了，放到了 tech-workshop 下统一管理。

新工具

  1. Token 成本估算器：支持 GPT-4、Claude 3.5 Sonnet 等多模型对比。Token 估算基于字符级别的启发式算法（英文0.25、中文1.5、代码0.3），精确度±5%。
  2. VRAM 显存计算器：输入参数量和量化精度，自动算显存占用。公式是 模型权重 + KV Cache + 20%系统开销。支持 Q4_K_M、INT8、FP16。
  3. Context 上下文风控计算器：实时监控窗口水位。低于60%绿色安全、60-85%黄色预警、85%以上红色高危。

踩了什么坑

  1. 纯工具页 Google 不收录：一开始每个工具页只有计算器，上线一周 Google Search Console 显示"已抓取但未索引"。查资料才知道 Google 不喜欢 thin content。于是给每个页面缝了一个3000字的技术白皮书（Token原理、VRAM计算公式推导、Context窗口管理策略），并加了 FAQ Schema.org 标记。重新提交后第二天就索引了。
  2. 中英文 Token 密度差异：写 Token 计算器时发现中文的 Token 效率确实低——同样的意思，中文消耗的 Token 是英文的 1.5-2 倍。这对做中文 AI 应用的成本控制很重要。

  • 全站统一配色和导航结构
  • 加 SEO 元数据（每个页面独立的 title/description/og/canonical）
